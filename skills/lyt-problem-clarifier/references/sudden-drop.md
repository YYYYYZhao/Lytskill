# 突然下滑

本文件用于「突然没单」「突然下滑」「之前能卖现在不行」。

## 核心判断

突然下滑，一定先找变化点，不要直接说平台限流。

## 追问模板

```text
突然下滑，先找变化点。

你先补 4 个信息：

1. 下滑从哪天开始？是曝光先掉、点击先掉、加购先掉、下单先掉，还是广告消耗先掉？
2. 下滑前 3 天你做过什么动作？改商品、改广告、改预算、改投入产出比目标、改素材、改库存、报活动、换达人、断货、改物流，有没有？
3. 同期有没有活动结束、竞品降价、评价变差、物流变慢、账号/商品提示？
4. 给我下滑前后各 3 天的曝光、点击、加购、下单、成交额、广告消耗。
```

## 优先问

如果用户只说「突然没单」，优先问：

- 曝光有没有掉。
- 点击率有没有掉。
- 加购/下单有没有掉。
- 广告消耗有没有掉。
- 最近动作是否影响链接权重、库存、价格、物流或素材。

## 禁止误判

- 不直接说平台限流。
- 不把单日波动当趋势。
- 不在没有下滑前后数据时判断某个动作导致结果。
- 不忽略活动结束、竞品降价、库存、物流和评价变化。
