# GMV Max 低消耗

本文件用于「广告跑不出去」「广告低消耗」「GMV Max 低消耗」。

## 默认口径

默认先按 TikTok Shop 商品成交额最大化广告，也就是 GMV Max 处理。

如果用户没有主动提异常提示，默认正常投放中，不问广告状态。

## 追问模板

```text
广告跑不出去或低消耗，默认先按 TikTok Shop 商品成交额最大化广告，也就是 GMV Max 处理；如果你没有提到异常提示，我先按正常投放中看。

你先补 5 个信息：

1. 广告建了多久？从几点开始跑，到现在实际跑了多长时间？
2. 当前日预算、实际消耗、曝光、点击分别是多少？
3. 投入产出比目标是多少？大概毛利率是多少，保本投入产出比有没有算过？
4. 选了几个产品？这些产品是新链接还是有历史成交/加购/点击？
5. 有没有可用视频素材？如果没有视频素材，商品卡自然流量有没有曝光、点击、加购、下单？
```

## GMV Max 规则

- 不按普通广告追问投放目标、出价方式、人群、地区定向。
- 核心可控项是预算、投入产出比目标、产品选择和素材选择。
- 用户已经明确没视频素材时，不要再问系统自动选素材。
- 只有用户主动说后台显示审核中、受限、拒审或其他异常提示时，才优先查审核和账户/商品资格。

## 预算消耗率

```text
预算消耗率 = 实际消耗 / 日预算
```

| 预算消耗率 | 追问方向 |
| --- | --- |
| 低于 50% | 跑不出去，先问投入产出比目标、商品基础盘、素材、链接历史数据 |
| 50%-80% | 有探索机会，问是否有订单、点击成本、产品集中度 |
| 高于 80% | 接近预算上限，问 ROI、订单稳定性、保本投入产出比 |

## 极弱启动信号

如果用户已经明确「没素材、商品卡没有流量、只选 1 个新品、24 小时无消耗」，追问顺序：

1. 商品卡/搜索流量：标题有没有覆盖买家会搜的核心词，商品卡自然曝光是否为 0。
2. 商品历史信号：新品、0 成交、0 加购、0 自然点击，系统没有历史反馈。
3. 投入产出比目标是否偏紧；不要把「降低投入产出比目标」写成唯一动作。
4. 素材缺口；没有视频素材时，补素材是增加内容流量渠道，但当前 0 消耗还要先看商品卡搜索流量。
