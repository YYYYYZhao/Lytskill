# 投入产出比差与利润问题

本文件用于「投入产出比差」「ROI 低」「成交额高但利润差」「利润薄」。

## 核心判断

投入产出比差，不等于广告一定差。要先看真实利润和订单稳定性。

## 追问模板

```text
投入产出比差，先别只看后台一个数。你先补 5 个信息：

1. 这是 GMV Max、商品广告，还是自然流量整体投入产出比？
2. 这段时间广告消耗、成交额、订单数、平均下单成本分别是多少？
3. 商品毛利率大概多少？保本投入产出比有没有算过？
4. 订单是不是集中在少数产品，还是很多产品都有消耗但没订单？
5. 最近退款、差评、物流问题有没有上升？
```

## 判断方向

| 现象 | 追问重点 |
| --- | --- |
| 消耗低，投入产出比高 | 样本太少，问预算消耗率、订单是否稳定 |
| 消耗高，投入产出比低 | 承接、价格、素材、人群、毛利 |
| 投入产出比高但成交额小 | 探索机会少，问预算、素材、产品数、目标是否太紧 |
| 成交额高但利润薄 | 毛利、退款、物流、佣金、广告真实成本 |
| 单子集中少数产品 | 产品分组、是否需要拆计划或单独观察 |
