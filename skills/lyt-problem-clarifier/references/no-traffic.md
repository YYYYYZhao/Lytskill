# 上品没流量与 0 曝光

本文件用于「上品没流量」「0 曝光」「商品搜索不到」「新品没展示」。

## 核心判断

先判断它是「不能展示」还是「能展示但没人点」。

```text
上品没流量，先判断它是「不能展示」还是「能展示但没人点」。

你先补 4 个信息：

1. 这个商品什么时候上架？现在后台商品状态是可售，还是审核中、拒审、暂停？
2. 后台有没有商品信息质量、提升商品信息质量、账号健康、商品诊断之类提示？
3. 上架后有没有改过类目、属性、标题、图片、品牌、物流模板、库存？
4. 现在累计曝光和点击是多少？是完全 0 曝光，还是有少量曝光但没人点？
```

## 完全 0 曝光

优先问：

- 商品状态
- 类目和属性
- 标题关键词
- 库存和物流模板
- 账号健康
- 商品诊断提示

不要直接让用户改主图。如果用户说后台有质量提示，先让他把提示内容贴出来。

## 有少量曝光

如果不是完全 0 曝光，而是有曝光但点击低，转读 `low-clicks.md`。

如果有曝光、有点击但不成交，转读 `no-orders.md`。

## 禁止误判

- 不把「没流量」直接解释成账号问题。
- 不把 0 曝光直接归因到主图。
- 不在商品状态未确认前建议加预算。
- 不跳过类目、属性、标题、库存、物流和商品诊断提示。
