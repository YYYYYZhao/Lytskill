# 成本与利润分析

本文件用于成本、利润、保本 ROI、退款、物流、平台费、佣金、优惠、售后和真实经营回报判断。

## 利润公式

```text
经营利润 = GMV - 货品成本 - 物流仓储 - 平台费用 - 达人佣金 - 广告费 - 优惠补贴 - 样品成本 - 人工运营 - 售后损耗
```

缺关键成本时只能做区间和情景假设，不能断言利润。

## 必查成本

| 成本项 | 看什么 |
| --- | --- |
| 货品成本 | 采购价、包装、质检、损耗 |
| 物流仓储 | 国内段、跨境段、本地尾程、仓储、退货 |
| 平台费用 | 类目佣金、交易手续费、平台支持费、活动费用 |
| 达人费用 | 标准佣金、广告佣金、样品和合作费 |
| 广告费用 | 实际消耗、平均下单成本 |
| 优惠补贴 | 优惠券、免邮、活动补贴 |
| 售后损耗 | 退款、退货、差评、丢件、补发 |

## ROI 处理

- ROI 不是利润。
- 平台 ROI 可能只反映归因成交额 / 广告消耗。
- 保本 ROI 要基于毛利率计算。
- 真实利润要把广告、物流、平台费、退款和优惠一起算。

```text
保本 ROI = 1 / 毛利率
```

毛利率不清时，先做三个情景：

| 情景 | 假设 | 作用 |
| --- | --- | --- |
| 保守 | 成本高、退款高、费用足额 | 看最差是否还能测 |
| 中性 | 使用当前最可能成本 | 看当前策略是否成立 |
| 乐观 | 成本优化、退款正常 | 看优化后天花板 |

## 判断口径

| 现象 | 判断 |
| --- | --- |
| GMV 高但利润薄 | 拆成本，不急着放量 |
| ROI 高但订单少 | 样本少，不能证明稳定赚钱 |
| 消耗高 ROI 低 | 查保本 ROI、成交成本、承接和商品 |
| 成本缺失 | 只能判断成交规模或流量效率 |
| 退款售后高 | 先查产品、履约和预期错配 |

## 利润动作

动作必须具体到对象和条件：

```text
对象：____
当前利润缺口：____
优先补的成本字段：____
可测试动作：改价 / 套装 / 运费 / 优惠 / 供应商 / 广告目标 / 停止投放
观察指标：利润、保本 ROI、平均下单成本、退款率、订单稳定性
继续条件：____
暂停条件：____
```

不要只写「提高 ROI」或「优化利润」。
