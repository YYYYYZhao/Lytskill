## 附件处理

### CSV 和 XLSX

1. 先列出工作表、行列数、表头和前几行。
2. 识别日期、对象标识、维度字段和指标字段。
3. 检查空值、重复、异常值、百分比、货币、时区和单位。
4. 确认分析粒度：商品、订单、广告组、素材、达人、日期还是渠道。
5. 使用结构化表格处理能力或脚本计算，不手工猜列。
6. 不改原始文件；需要时另存清洗版、汇总版或报告。

### 截图和图片

1. 先说明页面、时间范围、可见字段和筛选条件。
2. 把可见事实逐项提取出来。
3. 再写推断链和不能确定的部分。
4. 多张截图先检查是否属于同一时间范围和同一口径。
5. 截图裁切不全、数字模糊或缺少上下文时，明确标记限制。

### PDF 和报告

1. 先定位与用户问题相关的章节、表格和日期。
2. 区分原始数据、作者解释和二次推断。
3. 平台规则、费用和政策必须检查发布日期与适用国家。
4. 引用结论时注明页码或章节。

## 数据体检

| 检查项 | 要看什么 | 常见风险 |
| --- | --- | --- |
| 时间范围 | 几天，是否覆盖变化前后 | 把一天波动当趋势 |
| 样本量 | 曝光、点击、下单、订单是否足够 | 小样本 0 成交被误判 |
| 字段口径 | 点击率、转化率、订单、销量、GMV 定义 | 不同页面字段混算 |
| 分组维度 | 商品、广告组、素材、渠道、达人、日期 | 全店数据掩盖对象差异 |
| 缺失异常 | 加购、下单、成本、退款、利润是否缺失 | 只能判断流量，不能判断利润 |
| 对照组 | 历史、同批商品、竞品、渠道、素材版本 | 没基准却判断好坏 |

数据不足时，给出当前可以判断的部分和需要补充的字段，不硬编确定结论。

## 口径校验

| 易错字段 | 正确处理 |
| --- | --- |
| 销量 | 通常是售出商品件数，不一定等于订单数 |
| 订单数 | 下单次数；一个订单可能包含多件商品 |
| 客单价 | 成交额 / 订单数；没有订单数时不能精确计算 |
| 平均单件成交价 | 成交额 / 销量；不能替代客单价 |
| GMV | 成交额，不等于到账金额，也不等于利润 |
| ROI | 广告收入效率，不等于真实经营利润 |
| 广告归因 GMV | 可能包含自然、达人或系统归因订单 |
| 总趋势页 GMV | 判断总盘子，不与渠道页直接相加 |
| 商品价格范围 | 不代表主力成交价格，要看成交价格带 |

## GMV 四变量

```text
GMV = 曝光 × 点击率 × 点击成交率 × 客单价
```

- 点击率 = 点击 / 曝光。
- 点击成交率的分子可以是买家、订单或成交件数，但必须说明口径。
- 客单价 = GMV / 订单数；没有订单数时只能计算平均单件成交价。
- 如果源数据的「转化率」是曝光到成交，先说明口径。

| 组合现象 | 优先判断 | 下一步 |
| --- | --- | --- |
| 曝光少，点击也少 | 流量样本不足 | 查渠道、预算、链接历史、商品资格和素材 |
| 点击率低，有加购或成交 | 前端入口偏弱，后端有信号 | 改商品卡或素材入口，再补样本 |
| 点击率高，点击少，0 成交 | 前端信号可以，成交样本不足 | 补点击和后链路数据 |
| 点击足够，点击成交率低 | 承接偏弱 | 查价格、评价、详情、优惠、运费和信任 |
| 点击成交率高，曝光低 | 有信号但没放量 | 控制成本补曝光，验证稳定性 |
| 客单价低 | GMV 天花板低 | 做套装、组合包、加价购或阶梯优惠 |
| GMV 高但利润薄 | 成本吞利润 | 拆货品、物流、平台、佣金、广告和退款 |

## 小样本护栏

- 点击少于 50：不能因为 0 成交就判断商品、详情页或人群有问题。
- 曝光少于 500：只能看初始信号，不能稳定判断趋势。
- 先估算：`预期订单 = 点击数 × 参考点击成交率`。
- 预期订单小于 1 时，0 成交通常属于正常波动。
- 参考点击成交率优先使用用户历史数据或同批商品。
- 没有加购、下单和成交数据时，不直接把断点归因到详情页。

## 场景漏斗

```text
曝光 -> 点击 -> 停留/互动 -> 加购/咨询 -> 下单 -> 成交 -> 履约 -> 复购/好评
```

| 场景 | 关键链路 | 核心指标 |
| --- | --- | --- |
| 商品卡/搜索 | 搜索 -> 曝光 -> 点击 -> 加购 -> 下单 -> 成交 | 搜索词、曝光、点击率、加购率、点击成交率 |
| 短视频/推荐 | 推荐 -> 观看 -> 点击商品 -> 加购 -> 下单 -> 成交 | 播放、完播、商品点击率、加购、下单 |
| 直播 | 推荐 -> 进房 -> 停留 -> 点击商品 -> 下单 -> 成交 | 进房、停留、商品点击、讲品转化 |
| 达人 | 达人内容 -> 点击商品 -> 加购 -> 下单 -> 成交 | 达人数、层级、类型、点击和销售额 |

视频播放高但不出单时，先拆算法链、种草链和成交链，不默认详情页有问题。

## GMV Max 和广告

GMV Max 是商品增长调度器，不能照搬传统广告字段。

```text
商品基础盘 -> 标题/主图/价格/评价/库存/物流 -> 素材 -> 系统调度 -> 曝光 -> 点击 -> 加购 -> 下单 -> 成交 -> ROI
```

核心字段：

| 字段 | 用途 |
| --- | --- |
| 日预算、实际消耗、预算消耗率 | 判断能否跑出探索 |
| 投入产出比目标、实际 ROI、保本 ROI | 判断效率约束 |
| 订单数、平均下单成本 | 判断稳定性和成交成本 |
| 产品数、素材数 | 判断系统可调度空间 |
| 链接历史数据 | 判断是否已有点击和成交信号 |

```text
预算消耗率 = 实际消耗 / 日预算
保本 ROI = 1 / 毛利率
```

| 现象 | 判断方向 |
| --- | --- |
| 消耗低，ROI 高 | 样本少、目标偏紧或探索机会少 |
| 消耗低，ROI 正常 | 查商品基础盘、素材和历史信号 |
| 消耗高，ROI 达标 | 订单稳定时考虑逐步放量 |
| 消耗高，ROI 低 | 查承接、成本、产品和素材 |
| ROI 高，GMV 小 | 效率好看但规模小 |
| 单子集中少数产品 | 系统赛马，拆产品看贡献 |

## 成本与利润

```text
经营利润 = GMV - 货品成本 - 物流仓储 - 平台费用 - 达人佣金 - 广告费 - 优惠补贴 - 样品成本 - 人工运营 - 售后损耗
```

分析时至少检查：

- 货品成本、包装、质检
- 国内段、跨境段、本地尾程和仓储
- 类目佣金、交易手续费、平台支持费和活动费用
- 标准佣金、广告佣金、样品和合作费
- 广告实际消耗
- 优惠、免邮和活动补贴
- 退款、退货、差评、丢件和补发

缺关键成本时只能做区间和情景假设，不能断言利润。

## 竞品和第三方工具

正确顺序：

```text
明确问题 -> 选择资源条件相似的竞品 -> 提取数据 -> 区分事实/推断/假设 -> 设计验证
```

| 数据 | 正确用法 | 常见误判 |
| --- | --- | --- |
| 销量 | 判断售出件数 | 直接当订单数 |
| 销售额 | 判断成交规模 | 当利润 |
| 商品数占比 | 看上架结构 | 当成交结构 |
| 销量/销售额占比 | 看成交主力 | 忽略价格带 |
| 渠道占比 | 判断成交方式 | 推断单一打法 |
| 广告标识 | 说明可能有投放迹象 | 推出广告费和 ROI |
| 达人数量 | 看达人池规模 | 以为越多越好 |
| 达人销售额 | 看渠道贡献 | 直接得出达人利润 |

第三方工具页面口径不同，不直接相加；看到的数据只能证明页面展示的内容。

## 表格可读性

商品分析主表可使用：

```markdown
| 商品 ID | 商品名称（短） | 曝光 | 点击 | 点击率 | 加购 | 下单 | 点击成交率 | 客单价/单件成交价 | GMV | 判断 | 下一步 |
| --- | --- | ---: | ---: | ---: | ---: | ---: | ---: | ---: | ---: | --- | --- |
```

广告分析主表可使用：

```markdown
| 广告/产品 | 日预算 | 实际消耗 | 预算消耗率 | 曝光 | 点击 | 下单 | 成交额 | 实际 ROI | 保本 ROI | 平均下单成本 | 判断 | 动作 |
| --- | ---: | ---: | ---: | ---: | ---: | ---: | ---: | ---: | ---: | ---: | --- | --- |
```

规则：

- 同一张表使用统一指标、顺序和单位。
- 商品名保留源语言短原文，商品 ID、SKU 和货号完整保留。
- 正文引用商品使用：`商品 ID｜商品名称短原文｜中文说明`。
- 全空或没有信息增量的列可以省略，但要说明。

## 行动标准

每个动作至少写清：

```text
对象：改哪个商品、链接、素材、广告组或渠道。
动作：具体做什么。
改法：主图、标题、价格、素材、预算、达人或详情怎么改。
验证：看哪个指标和样本。
条件：什么情况下继续、暂停或换方案。
```

指标和动作必须对应：

| 弱变量 | 动作方向 |
| --- | --- |
| 曝光低 | 查渠道、预算、目标、链接历史、资格和素材 |
| 点击率低 | 改首图、首帧、标题前半段、价格表达和卖点顺序 |
| 加购低 | 查详情、规格、评价、价格、运费和购买理由 |
| 点击成交率低 | 查评价、优惠、发货、规格和信任 |
| 客单价低 | 做套装、组合包、加价购和阶梯优惠 |
| ROI 低 | 查保本 ROI、成交成本、毛利、退款和承接 |
| GMV 低 | 先拆四变量，再决定动作 |
